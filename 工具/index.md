# 微信

## 公众号文章中的视频如何保存下载

> **方法**

- 1.电脑端微信上鼠标右击公众号文章，在弹出的菜单栏中选择**使用默认浏览器打开**。
- 2.F12 打开开发者工具，在`Network`中筛选出`Media`。
- 3.然后点击播放目标视频。
- 4.右侧信息栏中会出现正在播放视频的信息，找到它（它后面是彩色的不断变化的）。
- 5.右键单击视频信息，并在弹出的菜单栏中选择`Open in new tap`选项。
  - 这里打开新网页有可能出现`访问 mpvideo.qpic.cn 的请求遭到拒绝 您未获授权，无法查看此网页。`的信息。在浏览器中刷新页面后再次操作即可。
- 6.此时该视频就会在新的网页中被打开，在视频的任意区域右键单击，并在弹出的菜单栏中选择`将视频另存为`。
- 7.弹出另存的对话框，选择存储位置，修改文件名，点击保存。
  - 若弹出另存为的对话框中，保存类型没有`.MP4`，可尝试在视频播放界面右下角设置处（三个点）点击选择`下载`。

> **参考链接**

- [微信公众号中的视频如何保存下来](https://zhuanlan.zhihu.com/p/338553003)

# B 站

## B 站视频字幕提取

> 以 B 站为例:
>
> - 只有视频有字幕（或 AI 字幕），才能用。
> - 无论是视频的字幕，还是 AI 字幕，都可以提取。比较简单。
> - 首先需要确定是视频的字幕还是 AI 字幕。如果是视频字幕，下面 F12 之后，输入的是 `json`，如果是 AI 字幕则需要输入 `ai_subtitle`。

> **方法**

- 1.打开视频
- 2.F12 打开开发者工具，在`Network`中筛选中输入`ai_subtitle`或者`json`。
- 3.刷新视频页面，出来请求后，点击请求，复制`Response`中所有内容。
- 4.打开[B 站字幕转换工具的网址](https://www.dreamlyn.cn/html/bsrt.html)，粘贴内容到输入框，点击`生成TXT`按钮，即可提取出字幕文本内容。也可点击`保存到本地`，保存的是`.srt`文件，可通过`记事本`读取文本内容。

- **写博客平台**

  - 如果你觉得上述的页面风格都太难看了，想动手搭一个自己喜欢的博客，那么可以使用 Hexo+GitHub 或者 WordPress。要想省事，还能别人家的平台(比如 Bitcron 等)，特点：如果你不做推广，基本没啥流量(没人会检索到你的文章)，算是写给自己看的。

  - 只想管理自己写过的文章，选择：GitHub/GitBook
  - 喜欢折腾：Hexo+GitHub 或者 WordPress

- **最适合程序员的笔记软件:**

`https://github.dev/[用户名]/[仓库名]`

它实际上就是 VS Code 编辑器的 Web 版，并且与 Git 高度集成。这种组合，还有什么笔记软件会比它更强！

使用时，首先在 GitHub 上面新建一个笔记仓库，公开或私密都可以。然后使用 `github.dev` 编辑完，再推送回仓库。换到其它机器时，就先从仓库获取文档的最新版本，简直是完美的解决方案。

更棒的是，GitHub 提供了一个快捷入口。 打开 GitHub 仓库主页，按一下小数点`（.）`这个键， 页面就会自动跳转到 VS Code 编辑环境，真是太方便了。

# 图床

> Q（gpt）：个人写博客，需要存储图片，有没有可以推荐的图床解决方案

A：

对于个人写博客并需要存储图片的情况，以下是一些常见的图床解决方案：

- 1. **第三方图床服务**： 有许多第三方图床服务提供免费或付费的图像存储和托管。常见的图床服务包括 Imgur、GitHub、七牛云、腾讯云 COS、阿里云 OSS 等。您可以选择适合您需求的服务，并按照其提供的接口或插件集成到您的博客中。

- 2. **自建图床服务器**： 如果您希望更加自主控制图床存储和管理，您可以考虑搭建自己的图床服务器。这需要您具备一定的服务器管理和维护经验。您可以使用类似 MinIO、Ceph 等开源工具来搭建私有的对象存储服务，以供您在博客中存储图片。

- 3. **云存储平台**： 如果您已经使用某个云服务提供商（如 AWS、Azure、Google Cloud 等），您可以使用其提供的云存储服务来存储和托管图片。这些云平台通常提供易于使用的 API 和工具，供您上传、管理和访问图片资源。

- 3. **本地存储**： 您也可以选择将图片直接存储在您的博客服务器上。这可以是您自己租用的虚拟私有服务器（VPS）或托管平台提供的服务器。在使用本地存储时，确保您的服务器具备足够的磁盘空间和带宽来支持图像存储和访问。

在选择图床解决方案时，**您需要考虑因素包括可靠性、性能、成本、易用性和数据安全等。根据您的需求和技术水平，选择适合您的图床方案**，并确保在使用第三方服务时了解其服务条款和隐私政策。
