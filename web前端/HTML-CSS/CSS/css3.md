## 其它
>【1】**space-around 和 space-evenly 的区别**

`space-evenly`，flex 元素之间和 flex 元素和容器的空间总是相等的，space-around，只有 flex 元素之间的空间是相等的，第一个 flex 元素和最后一个 flex 元素和容器的空间是 flex 元素间空间的一半，如下图。
